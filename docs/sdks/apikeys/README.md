# ApiKeys
(*environments.apiKeys*)

### Available Operations

* [list](#list) - Get api keys
* [regenerate](#regenerate) - Regenerate api keys

## list

Get api keys

### Example Usage

```typescript
import { Novu } from "novu/api";

const novu = new Novu({
  apiKey: "<YOUR_API_KEY_HERE>",
});

async function run() {
  const result = await novu.environments.apiKeys.list();

  // Handle the result
  console.log(result)
}

run();
```
